Butter

Butter is an image service where you use our api to auto generate URLs for images.

Use it to populate your working site.

Use it as a avatar suggestion.

Use it as a place holder.

APIs

Getting a image from persistent path:

https://butter.zzstudios.cn/p/{assigned-image-key}

Example:

https://butter.zzstudios.cn/p/COFFEE

(This is probably not an image of coffee. It just happens to have the key `COFFEE`. )
